# 检测识别

## 准备图像资源

为适配视觉模块的模型输入要求（支持320×240分辨率图像），当前处理流程如下：首先，通过CanMV平台接收来自K210开发板的摄像头图像数据，并将其编码为MP4视频文件；随后，利用MP4取帧工具将该视频逐帧提取为静态图像；最后，在堪智模型训练平台上对提取的图像进行标注，并完成模型训练。整个流程旨在实现从图像采集到模型部署的端到端数据准备与训练工作。

---

### 使用canmv录制视频

打开canmv ide，由于软件打开默认是中文，如需设置语言，请按照图示操作。设置完成后，需要重启软件生效。

<img src="./assets/image-20260624103216595.png" alt="image-20260624103216595" style="zoom:25%;" />

<img src="./assets/image-20260624103250369.png" alt="image-20260624103250369" style="zoom:25%;" />



将下面的代码复制到ide中。

```python
import sensor, image, time, lcd

lcd.init()                   # Init lcd display
lcd.clear(lcd.RED)           # Clear lcd screen.

sensor.reset(dual_buff=True)
sensor.set_pixformat(sensor.RGB565)
sensor.set_framesize(sensor.QVGA)   # 320x240
sensor.set_hmirror(1)
sensor.set_vflip(1)
sensor.set_auto_whitebal(False)
sensor.skip_frames(time=2000)

while(True):
    img = sensor.snapshot()         # Take a picture and return the image.
    lcd.display(img)                # Display image on lcd.
```

# 分类识别

## 准备图像资源

为适配视觉模块的模型输入要求（支持128×128分辨率图像），当前处理流程如下：首先，通过CanMV平台接收来自K210开发板的摄像头图像数据，并将其编码为MP4视频文件；随后，利用MP4取帧工具将该视频逐帧提取为静态图像；最后，在堪智模型训练平台上对提取的图像进行标注，并完成模型训练。整个流程旨在实现从图像采集到模型部署的端到端数据准备与训练工作。

### 使用canmv录制视频

将下面的代码复制到ide中。

```python
import sensor, image, time, lcd

lcd.init()                   # Init lcd display
lcd.clear(lcd.RED)           # Clear lcd screen.

sensor.reset(dual_buff=True)
sensor.set_pixformat(sensor.RGB565)
sensor.set_framesize(sensor.B128X128)   # 128x128
sensor.set_hmirror(1)
sensor.set_vflip(1)
sensor.set_auto_whitebal(False)
sensor.skip_frames(time=2000)

while(True):
    img = sensor.snapshot()         # Take a picture and return the image.
    lcd.display(img.resize(230, 230))                # Display image on lcd.
```

步骤参考 `检测识别 - 使用canmv录制视频` 章节。

因为使用的是128\*128的分辨率，所以右侧回传画面较小。

> 注意：与检测识别不同，分类识别不需要标注，检测时仅能反应目标物体在当前图像中的概率。 所以在录制视频时，应该保持目标物体的背景干净，没有杂物。
